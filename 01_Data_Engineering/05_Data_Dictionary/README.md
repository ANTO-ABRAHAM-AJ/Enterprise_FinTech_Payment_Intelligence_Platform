# Data Dictionary

## Overview

The Data Dictionary documents every table and column used in the Enterprise FinTech Payment Intelligence Platform.

It provides:

- Table descriptions
- Column names
- Data types
- Business meaning
- Primary Keys (PK)
- Foreign Keys (FK)

## Files

- 01_Data_Dictionary.xlsx
- 02_Data_Dictionary_Explanation.md

## Purpose

A data dictionary improves data governance, documentation, and collaboration by providing a single source of truth for all database objects.
