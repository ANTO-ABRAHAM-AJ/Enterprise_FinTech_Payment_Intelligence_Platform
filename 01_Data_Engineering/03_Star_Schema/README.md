# Star Schema

## Enterprise FinTech Payment Intelligence Platform

This folder contains the Star Schema design of the Enterprise FinTech Payment Intelligence Platform.

## Components

- Fact_PaymentTransactions
- Dim_TransactionType
- Dim_Fraud

## Purpose

The Star Schema organizes transactional payment data into a centralized Fact table and supporting Dimension tables. This design improves analytical query performance, reduces redundancy, and simplifies reporting for business intelligence and Power BI dashboards.
